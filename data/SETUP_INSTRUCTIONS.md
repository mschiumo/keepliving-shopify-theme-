# KeepLiving Shopify Store Setup Guide

## Step 1: Import Products

1. Go to Shopify Admin → Products → Import
2. Click "Import"
3. Upload `data/products.csv`
4. Review and confirm import

**Note:** Products will be imported as drafts with placeholder images.

## Step 2: Upload Product Images

Option A: Use Printful (Recommended)
- Create products in Printful
- They'll automatically sync to Shopify with mockups

Option B: Manual Upload
1. Shopify Admin → Content → Files
2. Upload all product images
3. Edit each product and add real image URLs

## Step 3: Configure Theme

1. Go to Online Store → Themes
2. Find "KeepLiving Custom Theme" (should already be uploaded)
3. Click "Customize"
4. Configure sections:
   - **Hero:** Update headline, description, button
   - **Featured Products:** Select your product collection
   - **About:** Add your personal story

## Step 4: Set Up Printful Integration

1. Install Printful app from Shopify App Store
2. Connect your Printful account
3. Sync products (or create new ones in Printful)
4. Set up shipping rates

## Step 5: Configure Store Settings

- **Settings → General:** Store name, email, address
- **Settings → Payments:** Connect payment provider
- **Settings → Shipping:** Set up shipping zones
- **Settings → Taxes:** Configure tax settings
- **Settings → Notifications:** Customize email templates
