# KeepLiving Shopify Theme

## Brand Identity
- **Mission:** A reminder to overcome fear, depression, hopelessness — make it through that moment/day/hard time.
- **Mood:** Calm but strong, minimalist, readable, authentic
- **Colors:** 
  - Primary: Deep teal/blue (#2c5f5d) — trust, calm
  - Accent: Warm coral/orange (#e07a5f) — hope, energy
  - Neutral: Soft white, light gray
- **Typography:** Clean sans-serif for readability

## Theme Structure
```
keepliving-shopify/
├── layout/
│   └── theme.liquid          ✓ Main layout wrapper
├── templates/
│   ├── index.json            ✓ Homepage

│   ├── collection.liquid     ✓ Collection page

├── sections/
│   ├── header.liquid         ✓ Site header with nav
│   ├── footer.liquid         ✓ Site footer

│   ├── featured-products.liquid ✓ Product grid
│   └── about.liquid          ✓ Story/About section
├── snippets/
│   ├── meta-tags.liquid      ✓ SEO meta tags
│   └── cart-badge.liquid     ✓ Cart count
├── assets/
│   ├── theme.css             ✓ Main stylesheet (comprehensive)
│   └── theme.js              (JavaScript)
├── config/
│   └── settings_data.json    ✓ Theme presets
└── README.md
```

## Installation Instructions
1. Zip the `keepliving-shopify` folder
2. In Shopify Admin: Online Store → Themes → Upload theme
3. Customize sections via theme editor
4. Add products and connect Printful
5. Configure domain

## Customization
Edit sections in Shopify theme editor:
- Colors: Settings → Colors
- Typography: Settings → Typography
- Homepage sections: Sections tab
- Products: Products → All products
